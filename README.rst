# WBZ and PIC32WM Prebuilt Firmware and Source Packages

This repository provides **prebuilt firmware binaries** and **reference source code** for **Microchip WBZ** and **PIC32WM** series wireless devices.  
It enables developers to **evaluate**, **flash**, and **rebuild** Zephyr- and H3-based wireless applications for BLE, Zigbee, and multi-protocol use cases.
